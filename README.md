# Overwatch 2 パーク配置クイズ

Overwatch 2の各キャラクターのパークが、ゲーム内で左右どちらに配置されているかを当てるクイズアプリです。

## 機能

- 🎮 全ヒーローのパークを網羅（タンク、ダメージ、サポート）
- 📊 リアルタイムスコア表示
- 🎯 正解・不正解のカウント
- 🔄 問題のランダム出題
- 📱 レスポンシブデザイン（PC・スマホ対応）
- 🇯🇵 完全日本語対応

## 使い方

1. `index.html`をブラウザで開く
2. キャラクター名とパーク名が表示されます
3. そのパークが左右どちらに配置されているか選択
4. 正解・不正解が表示され、スコアが更新されます
5. 「次の問題」ボタンで次の問題へ進む
6. 「リスタート」ボタンでスコアをリセット

## ファイル構成

```
ow-park-quiz/
├── index.html   # メインHTMLファイル
├── style.css    # スタイルシート
├── data.js      # ヒーローとパークのデータ
├── app.js       # クイズロジック
└── README.md    # このファイル
```

## スコアリング

- ✅ 正解: +10ポイント
- ❌ 不正解: 0ポイント

## 対応ブラウザ

- Chrome（推奨）
- Firefox
- Safari
- Edge

## カスタマイズ

`data.js`ファイルを編集することで、ヒーローやパークを追加・変更できます。

```javascript
{
    hero: "キャラクター名",
    perks: [
        { name: "パーク名", position: "left" },  // 左配置
        { name: "パーク名", position: "right" }  // 右配置
    ]
}
```
